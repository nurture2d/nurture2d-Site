System Requirements

What you need to run Nurture2D.

Tested on real hardware — from a comfortable daily driver to a modest entry-level machine. These are honest results, not marketing benchmarks.

Recommended

Smooth Experience

Tested on Linux — runs without any noticeable issues.

  • Resolution 1920 × 1080
  • RAM 8 GB
  • Storage 512 GB NVMe SSD
  • GPU AMD Ryzen 5 Integrated Graphics
  • OS Linux (recommended)
Minimum

Usable, with caveats

Tested on Windows — functional, though performance feels slightly sluggish on heavier projects.

  • Resolution 1366 × 768
  • RAM 4 GB
  • Storage 256 GB NVMe SSD
  • GPU Intel UHD 600 Integrated Graphics
  • OS Windows 10 / 11

These results reflect real-world testing conditions and are not guaranteed benchmarks. Performance will vary depending on project complexity, background processes, and system configuration. Notably, Linux tends to be significantly lighter on resources than Windows, which partly accounts for the difference in experience between the two test machines. The UI and layout are designed to accommodate both the recommended and minimum resolutions.
