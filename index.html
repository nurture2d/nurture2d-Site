Features

Everything an animator needs.
Nothing they don't.

Built from the ground up for traditional 2D workflows — from rough sketches to clean lineart to final color.

Paper-like Drawing

Responsive brushes, onion skinning, layers, and pressure curves that feel like real pencil on paper.

Frame-by-Frame Control

Accurate frame timing, exposure control, and flipbook playback.

Auto Color Fill

Intelligent fill that works with non-antialiased lines — perfect for cell shading and anime-style art.

Switchable Color Wheels

Toggle between square and triangle wheels. Popup color picker at your cursor for clean workflow.

Focused Interface

A clean, distraction-free UI that keeps your canvas front-and-center. Hide everything with a single Tab keypress.

Export with Confidence

Render to image sequences in separate folders, ready for production and post-processing.

Advanced Curve Tools

Professional spline drawing tools similar to Clip Studio Paint for precise, expressive line creation.

Reference Window

Dedicated reference panel for design sheets, color palettes, and sending current drawings as reference.

Drawing Tools

Built for the way animators actually draw.

Five pen presets optimized for no-antialiasing, a custom sketch brush engine with natural pencil textures, and a draw-behind feature for complex layering. Every tool is designed around the traditional animation workflow.

  • Pen Tool presets with color optimization
  • Custom sketch brush with pressure curves
  • Draw behind existing elements
  • Onion skinning with shift & trace
Color & Fill

Color that works with your lines, not against them.

The auto color line fill system intelligently fills non-antialiased preset lines. Fill transparent pixel gaps with lasso and rectangle selection. Prevent accidental fills of lineart with intelligent color detection.

  • Auto Color Line fill for cell shading
  • Transparent gap filling tools
  • Fill tools ignore target color
  • Dplate tool for character color sheets
